# Minimal Hyprland Dotfiles (Retro-Safe)

Minimal Arch Linux dotfiles focused on:
- Hyprland
- tmux
- LazyVim
- Waybar (minimal)
- Rofi
- Alacritty
- Nemo file manager

Designed to run on **low-RAM systems **.

---

## Included
- Hyprland (Wayland)
- Waybar (minimal config)
- Rofi launcher
- Alacritty terminal
- tmux
- LazyVim (Neovim)
- Nemo file manager
- fastfetch (manual use)

---
